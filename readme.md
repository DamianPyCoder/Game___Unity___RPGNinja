# RPG NINJA GAME 2D
[Download code](https://github.com/DamianPyCoder/Unity__Games__x6/tree/main/Code_RPG)

### How to play:
```
   Move left:          A
   Move right:         D
   Move down:          S
   Move up:            W
   Select or Attack:   Mouse click
```
